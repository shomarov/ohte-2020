# Software requirements

## Overview

With this software users are able to scan for and manage duplicate photos.

## Functionality

- user is able to register and login with username/password combination
- user is able to scan hard drive or folder for photos
- user is able to view the results of a scan
- user is able to scan hard drive or folder for duplicate photos
- user is able to view results of a duplicate photo scan
- user is able to delete selected photos
- user is able to view single photo's metadata
